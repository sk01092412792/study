# HSE 공급업체 평가 시스템 (HSE Supplier Evaluation System)

이 프로젝트는 공급업체의 보건(Health), 안전(Safety), 환경(Environment) 관리 능력을 평가하기 위한 웹 기반 체크리스트 시스템입니다.

## 📝 프로젝트 개요

이 시스템은 PSA Busan 또는 관련 기관이 공급업체를 대상으로 ISO 규정 준수 여부 및 안전 관리 정책을 평가할 수 있도록 돕습니다.
사용자는 웹 폼을 통해 각 항목에 답하고, 제출 시 자동으로 점수가 계산되어 결과를 확인할 수 있습니다.

## 📂 주요 파일 설명

- **`index.php`**: 애플리케이션의 핵심 파일입니다. 사용자 인터페이스(HTML 폼)와 점수 계산 로직(PHP)이 포함되어 있습니다.
- **`style.css`**: 웹 디자인을 담당하는 스타일시트입니다.
- **`ABB Contractor HSE evaluation form_rev.03.xlsx`**: 평가 항목의 기준이 되는 엑셀 파일입니다.

## 🚀 사용/실행 방법

1. `index.php` 파일이 있는 경로에서 PHP 서버를 실행합니다.
2. 웹 브라우저로 해당 주소에 접속합니다.
3. 회사명, 대표자명 등을 입력하고 각 체크리스트 항목에 답변합니다.
4. "평가 제출" 버튼을 누르면 점수가 계산되어 표시됩니다.

## 🛠 기술 스택

- **Frontend**: HTML5, CSS3 (Custom Styling)
- **Backend**: PHP (Form handling & logic)
